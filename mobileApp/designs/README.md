# Astra Salon — Design source files

Drop the exported Claude design HTML here, using these exact filenames. Claude
reads them from this folder to extract tokens/components and build the Flutter UI.

## Master theme / design system
- `Astra Salon - Signature.dc.html`   ← master tokens (typography, spacing, radius, shadows, components)

## Screens
- `Astra Salon - Sale Flow.dc.html`   ← POS sale flow
- `Astra Salon - Admin.dc.html`       ← admin dashboard/home
- `Astra Salon - Reports.dc.html`     ← reports (bill-wise / employee-wise)
- `Astra POS - Premium.dc.html`       ← premium POS layout variant

## Color presets (Settings → theme picker)
- `Astra Salon - Sage.dc.html`
- `Astra Salon - Slate.dc.html`
- `Astra Salon - Aqua.dc.html`
- `Astra Salon - Premium.dc.html`
